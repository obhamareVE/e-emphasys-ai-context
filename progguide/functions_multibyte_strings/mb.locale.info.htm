mb.locale.info()
Syntax:

function string  mb.locale.info  (long info_flag)

Description

This function returns information about the current locale.

Arguments
long info_flag

Specifies the required information. The possible values are:

TSS_GET_LOCALE_NAME returns the locale name
TSS_GET_NLS_NAME returns the NLS name
TSS_GET_TSS_NAME returns the name of the character set
TSS_GET_IFACTOR returns the internal mb factor
TSS_GET_EFACTOR returns the external (database) factor

 

 

Return values

A string containing the requested information. Where applicable, lval() can be used to convert the string to a long.

Context

This function is implemented in the porting set and can be used in all script types.
