mb.strpos()
Syntax:

function long  mb.strpos  (string string_expr, long pos)

Description

This function converts a screen position to a string position. Code features are ignored.

Arguments
string string_expr

The source string.
long pos

A screen position.

 

Return values

The string position corresponding to the specified screen position. Or -1 if pos lies beyond the end of the string.

Context

This function is implemented in the porting set and can be used in all script types.
