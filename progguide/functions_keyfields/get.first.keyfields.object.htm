get.first.keyfields.object()
Syntax:

function long get.first.keyfields.object  (long object)

Description

Get the first key fields object in this collection.

Arguments
long object

ID of the key fields collection which must be returned by a previous call to create.keyfields.collection()

 

Return values
<>0 The id of the first key fields object
0 When this collection does not contain any objects

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
