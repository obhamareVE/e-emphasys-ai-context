is.field.multi.currency()
Syntax:

function boolean  is.field.multi.currency  (string field_name(18))

Description

This returns whether a specified field is a multi currency field.

Arguments
string field_name(18)

 

Return values
TRUE Field is a multi currency field.
FALSE Field is not a multi currency field.

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
