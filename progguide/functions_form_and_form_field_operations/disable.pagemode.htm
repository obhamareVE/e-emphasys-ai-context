disable.pagemode()
Syntax:

function void  disable.pagemode  ()

Description

This disables pagemode for this session, even when the user runs in pagemode.

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types.

Note

This function is deprecated, as page mode is no longer supported.
