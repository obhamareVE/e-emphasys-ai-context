sel.num.selected()
Syntax:

function long  sel.num.selected  ()

Description

This function returns the number of records that is selected in the session.

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types. This function is available from TIV level 1075.

Note

This function is avialable from Tools Interface Version (TIV) TIV 1075.
