ml_get_datalang()
Syntax:

function string  ml_get_datalang  ()

Description

Returns the current data language of the user.

Return values

the current data language of the user, or "" in case Multi Language support is inactive

Context

This function is implemented in the 4GL Tools and can be used in all script types.
