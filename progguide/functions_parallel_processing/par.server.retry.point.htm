par.server.retry.point()
Syntax:

function void par.server.retry.point  ()

Description

This function is used for setting a not real retry point after a db.retry.point (). It must be used together with function par.server.retry.hit(). See section: Parallel Application Processing Database Retries for a further explanation.

Context

This function is implemented in the 4GL Engine and can be used in 3GL script types.
