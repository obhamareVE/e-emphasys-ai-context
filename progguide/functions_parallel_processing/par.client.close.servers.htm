par.client.close.servers()
Syntax:

function boolean par.client.close.servers  (long group.id)

Description

This function will close all servers in a group. Normally this call should only be done when all servers are ready, so after a call to par.client.wait.ready(). Closing the servers means that the application servers will terminate.

Arguments
long group.id

ID which is returned by a previous call to par.init.client.to.server().

 

Return values
true When successful
false When an error occurred

 

Context

This function is implemented in the 4GL Engine and can be used in 3GL script types.
