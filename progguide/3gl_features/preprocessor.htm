Preprocessor

The bic6.2 compiler for 3GL program scripts has a built in preprocessor with the following functions:

  • Include files (preprocessor)
  • Macro definition (preprocessor)
  • Object identifications (preprocessor)
  • Pragma codes (preprocessor)
  • Conditional compiling (preprocessor)

All preprocessor directives starts with a '#'. A preprocessor directive must be the first word on a source line, possibly with leading spaces or tabs.
