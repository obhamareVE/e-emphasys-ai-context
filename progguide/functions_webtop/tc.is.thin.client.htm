tc.is.thin.client()
Syntax:

function boolean  tc.is.thin.client  ()

Description

This function returns whether you are running in thin-client mode.

Return values

true          running in thin-client mode

false         NOT running in thin-client mode

Context

This function is implemented in the 4GL Engine and can be used in all script types.
