is.image.changed()
Syntax:

function boolean is.image.changed  (string fieldname)

Description

This function can be used to check if an image has been changed onto the image field or the image is changed which is not yet saved or discarded.

Arguments
string fieldname

The name of the image field on the form.

 

Return values
true When an image is changed for this image field.
false When no image is changed for this image field.

 

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types.
