destroy.tree()
Syntax:

function void  destroy.tree  (string tree_name [, long process_id])

Description

Deprecated. This function is only supported for Baan Windows and its usage is therefore deprecated.

This ends the Structure Chart Manager for the specified tree.

Arguments
string tree_name

The name of the tree that you want to kill.
[long process_id]

This optional argument indicates the process ID of the tree that you want to end, as returned by create.tree(). If you do not include this argument, all trees with the name tree_name are killed.

 

Context

This function is implemented in the 4GL Tools and can be used in 4GL script types.
