Programmable dialogs synopsis
Overview

Use these functions for programmable dialogs.

Synopsis
long dialog.new() ( string dialogName() [, long attribute, value] ... )
long dialog.add.field() ( long dlg, const string fldName(), const string fldLabel() [, long attribute, value] ... )
long dialog.add.listbox() ( long dlg, const string fldName(), const string fldLabel(), long no.items, long enum.vals(), string enum.desc(,) [, long attribute, value] ... )
long dialog.add.text() ( long dlg, const string text() [, long attribute, value] ... )
long dialog.add.button() (long dlg, const string btnFunction(), const string btnLabel())
long dialog.show() ( long dlg [, long previous_window, long load_defaults] )
long dialog.refresh.field() ( long dlg, const string fldName() )
long dialog.add.chart() ( long dlg, long initial.type [, long allowed.types] )
long chart.set.title() ( long chart, const string title [,const string subtitle] )
long chart.set.axis.type() ( long chart, long axis, long type )
long chart.set.axis.title() ( long chart, long axis, const string title )
long chart.add.series() ( long chart , const string title )
long chart.add.line.series() ( long chart , const string title )
long chart.add.data.point() ( long series , (long|string|double) x, (long|double) y )
long chart.clear.data() ( long chart )

 

Note

Grids are not supported by programmable dialogs.
