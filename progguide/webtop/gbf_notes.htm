Some notes on the Generic Browser Framework (GBF)

The following GBF options are not supported:

Option flag Description
GBF.OPT.2.WINDOWS

Have 2 windows:

left one for interior objects in tree format

right one for leaf nodes in free format
GBF.OPT.TREE.CONTROL Use local windows Tree Control for drawing the actual tree
GBF.OPT.DRAG.DROP Main application supports drag and drop (see gbf.drag.drop()
