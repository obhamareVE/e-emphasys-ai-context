dbcm.is.action.active()
Syntax:

function boolean dbcm.is.action.active  ()

Description

Tests whether the currently selected object action (as selected with dbcm.select.object.action()) is active according to the Object Configuration Management deployment.

Return values
true The currently selected object action is active.
false In any other case.

 

Context

This function is implemented in the 4GL Engine and can be used in all script types. This function is available from TIV level 1800.
