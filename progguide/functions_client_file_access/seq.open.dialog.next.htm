seq.open.dialog.next()
Syntax:

#include <bic_desktop>

function long seq.open.dialog.next  (ref string filename)

Description

Retrieve the next full pathname of the file selected by the user in a previous seq.open.dialog.local() function. This function should only be used when a previous call to seq.open.dialog.local() returned a value greater than 1.

Arguments
ref string filename

Output argument which will contain the next full path of the file selected by the user.

 

Return values
true Successfully retrieved next filename
false No more filenames to be retrieved

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Notes

This function is not supported in HTMLUI. See the Implementing HTML UI support for more information.
