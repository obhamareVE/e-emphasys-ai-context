client.show.url
Syntax:

#include <bic_desktop>

function long client.show.url  (string url)

Description

This function opens a URL in a Document Viewer session. This function is only supported in the WebUI.

Arguments
string url

This specifies the URL that needs to be started.

 

Return values
0 success
<0 when an error occured (for instance invalid source path)

 

Context

This function is implemented in the 4GL Engine and can be used in 3GL script types.
