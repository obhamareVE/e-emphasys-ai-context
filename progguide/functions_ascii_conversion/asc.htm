asc()
Syntax:

function long asc  (string characters)

Description

This function returns the sum of the code point values encoded in the supplied string.

If the supplied string is a single byte string, then each byte of the supplied string is treated as a separate code point, so effectively the sum of all the bytes of the string is returned.

If the supplied string is a multi byte string, then a byte with hexadecimal value 0x9B is treated as the beginning of a four-byte encoding of a TSS Encoding character 9B pp qq rr, which contributes the hexadecimal value 9B ppqqrr to the value to be returned. Other bytes contribute their own value to the value to be returned.

Arguments
string characters

 

Context

This function is implemented in the porting set and can be used in all script types.

Note

Multibyte characters with very high values can cause the returned long to overflow and loose its meaning. No check is made on the legality of the value.

Example
string  select(10)
asc_b = asc("B")                | asc_b is set to 66
asc("abc")                      | Returns 294 (that is, 97 + 98+ 99)
asc(select)                     | Returns the sum of the ASCII codes for each character 
                                | in the string variable select
