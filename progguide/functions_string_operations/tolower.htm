tolower$()
Syntax:

function string  tolower$  (string str)

Description

This function converts all upper case characters in its argument to lower case. It returns the converted string expression. The input string remains unchanged.

To convert all lower case characters to upper case, use toupper$() instead.

Arguments
string str

 

Context

This function is implemented in the porting set and can be used in all script types.

Note

Thise function can handle multibyte characters.
