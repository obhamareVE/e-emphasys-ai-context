string.set$()
Syntax:

function string  string.set$  (string string_expr, long num_expr)

Description

This returns a string that contains string_expr repeated num_expr times.

Context

Bshell function.

Arguments
string string_expr
long num_expr

 

Context

This function is implemented in the porting set and can be used in all script types.

Example
string st(80)
st = string.set$("-", 10)          | st contains "----------"
st = string.set$( string.set$("-",3)&"+", 3 )
                                   | st contains "---+---+---+"
st = string.set$( chr(135), 5 )    | st contains 5 graphic symbols
