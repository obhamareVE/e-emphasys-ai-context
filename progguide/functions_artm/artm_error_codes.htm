Application Response Time Measurement (ARTM) Error Codes
-1000 ARTM is not active or a non existing level is passed in the function artm.define.transaction.class()
-1001

One of the following occurred:

  • An invalid metric type is passed in the metric list
  • ARTM.SKIP.METRIC is passed in artm.define.transaction.class()
  • An invalid transaction status is passed.
-1002 Too many metrics have been passed.
-1003 A metric type has been defined, but the name or the value is missing.
