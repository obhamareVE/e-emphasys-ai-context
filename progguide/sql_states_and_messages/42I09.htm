42I09 - Combined column in indexing or substring expression

An SQL statement contains an expression in which an array indexing operator or a substring operator is applied to a combined column.

Example
select *
from   dbtst120
where  cmbb(1) > 50000

The column cmbb is a combined column that contains two child columns salary and bonus. The expression cmbb(1) will cause the error:

SQLState 42I09: Combined column 'cmbb' in indexing or substring expression
