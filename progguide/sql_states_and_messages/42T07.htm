42T07 - Invalid type for array index

An SQL statement contains an array indexing expression and the index operand of this expression is not of the correct type.

Example
select firstnme
from   dbtst120
where  phoneno(bonus) = "0342"

The index operand of an indexing expression must be of type integer. Because column bonus is of type real, this query will cause the error:

SQLState 42T07: Invalid type for array index. Expected type INTEGER
