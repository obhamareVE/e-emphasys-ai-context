42T01 - Incompatible types for comparison operator

An SQL statement contains a comparison predicate and the operands of this comparison predicate do not have comparable datatypes.

Example
select empno
from   dbtst120
where  edlevel = "CHRISTINE"

The column edlevel is of type integer. Integer and string are not comparable datatypes. This query will cause the error:

SQLState 42T01: Incompatible types for comparison operator '='
