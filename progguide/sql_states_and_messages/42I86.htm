42I86 - Invalid enum description

An argument of the enum_description function is invalid.

Example
select enum_description( sex, '12' )
from dbtst120

Because the language argument is invalid, this query will cause the error:

SQLState 42I86: The length of the language specification in an enum_description function must be exactly 1

In the following example, the first argument is not a valid column name:

select enum_description( 'state', '2' )
from dbtst120

This query will cause the error:

SQLState 42I86: The first operand of the enum_description function must identify a column of a table name
