42I75 - RETRY clause cannot be used in an ORDER BY clause that references a generic value expression

An SQL statement contains an ORDER BY clause with a RETRY clause and one of the columns specified by the ORDER BY clause is a _compnr column or a generic value expression that is not a simple column reference.

Example
select firstnme, _compnr
from   dbtst120
order  by 1,2 with retry

The ORDER BY clause specifies that the result must be ordered by column firstnme and _compnr. If the ORDER BY clause contains a RETRY clause then it may not reference a _compnr column. This query will cause the error:

SQLState 42I75: RETRY clause cannot be used in an ORDER BY clause that references a generic value expression
