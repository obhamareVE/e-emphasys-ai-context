42I63 - Combined column used in SET clause

An SQL UPDATE statement contains a combined column with more than one child column in its SET clause.

Example
update dbtst120
set    cmbb = 50000
where  edlevel > 0

Column cmbb is a combined column that contains two child columns salary and bonus. This statement will cause the error:

SQLState 42I63: Combined column 'cmbb' used in SET clause

A combined column that contains just one child column can be used in the SET clause.
