42I06 - Array indexing operator applied to non array column

An SQL statement contains a an array indexing operator that is applied on a column that is not an array column.

Example
select *
from   dbtst120
where  workdept(1,1) > 20000

Column workdept is not an array column. The expression workdept(1,1) will cause the error:

SQLState 42I06: Array indexing operator applied to non array column
