42I74 - A RETRY clause cannot be used in a UNION query

A UNION query contains a RETRY clause.

Example
select respemp
from   dbtst160
union  all
select mgrno
from   dbtst100
order  by 1 with retry

The ORDER BY clause of a UNION query may not contain a RETRY clause. This query will cause the error:

SQLState 42I74: A RETRY clause cannot be used in a UNION query
