42I47 - Order by on array column

An SQL statement contains an ORDER BY clause that contains an array column or an ORDER BY number that identifies an array column.

Example
select firstnme, lastname, phoneno
from   dbtst120
order  by 3

An ORDER BY cannot be performed on an array column. Column phoneno is an array column, therefore this query will cause the error:

SQLState 42I47: Order by on array column
