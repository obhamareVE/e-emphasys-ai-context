42T03 - Incompatible types for IN operator

An SQL statement contains an IN predicate and the operands of this IN predicate do not have comparable types.

Example 1
select empno
from   dbtst120
where  workdept in ( 10, 20 ,30 )

Column workdept is of type string. String and integer are not comparable datatypes. This query will cause the error:

SQLState 42T03: Incompatible types for IN operator

Example 2
select empno
from   dbtst120
where  workdept in ( select mgrno
                     from   dbtst100 )

Column workdept is of type string. Column mgrno is of type integer. String and integer are not comparable datatypes. This query will cause the error:

SQLState 42T03: Incompatible types for IN operator
