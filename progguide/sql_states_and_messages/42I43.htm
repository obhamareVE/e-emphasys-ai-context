42I43 - Maximum number of index hints per table exceeded

An SQL statement contains an index hint that specifies more index numbers than is currently allowed.

Example
select *
from   dbtst120, dbtst100
where  workdept refers to dbtst100
hint   use index 1,2,3,4,5 on dbtst100

The maximum number of indexes that can be specified with one index hint is 4. This query causes the error:

SQLState 42I43: Maximum number of index hints per table (4) exceeded
