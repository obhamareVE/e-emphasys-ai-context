42I23 - ORDER BY column is not in SELECT clause

The ORDER BY clause of an SQL statement contains a column that is not in the SELECT clause of that statement.

Example
select firstnme, lastname
from   dbtst120
order  by workdept

An ORDER BY clause can only operate on columns that are selected. Because Column workdept is not selected this query will cause the error:

SQLState 42I23: ORDER BY column 'workdept' is not in SELECT clause

An ORDER BY column does not have to be explicitly listed in the SELECT clause, just as long as it is selected. For example, the following query is correct even though the column salary is not explicitly listed in the SELECT clause. 

select *
from   dbtst120
order  by salary
