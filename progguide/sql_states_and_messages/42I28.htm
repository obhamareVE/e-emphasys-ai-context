42I28 - Unknown enum value

An SQL statement contains a reference to an enum constant that does not exist in the runtime data dictionary.

Example 1
select firstnme
from   dbtst120
where  sex in ( dbsex.male, dbsex.female, dbsex.unknown )

The enum value dbsex.unknown does not exist. This query causes the error:

SQLState 42I28: Unknown enum value 'dbsex.unknown'

Example 2
select firstnme, deptname
from   dbtst120, dbtst100
where  workdept refers to dbtst100
and    empno in ( dbtst100.mgrno, 100, 200 )

A set of values of an IN predicate cannot contain column references. Every identifier in a set of values is assumed to be an enum value. Therefore, this query will cause the error:

SQLState 42I28: Unknown enum value 'dbtst100.mgrno'
