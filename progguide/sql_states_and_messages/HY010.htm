HY010 - Function sequence error

The SQL CLI (Call Level Interface) dictates a certain order of interface calls. This error is issued when this constraint is violated.

Example
table tdbtst120

function main()
{
   long id

   id = sql.parse("select firstnme from dbtst120 where empno = 10")
   sql.fetch(id)
   sql.close(id)
}

The sql.fetch() call may only be done after a sql.exec() call. This program will cause the error:

SQLState HY010: Function sequence error - Fetch called out of sequence
