42I30 - Compnr specification in OR condition

An SQL statement contains a company number specification that is contained in an OR condition.

Example
select *
from   dbtst120
where  empno > 100 or ( empno < 100 and _compnr = 100 )

This query causes the error:

SQLState 42I30: Compnr specification in OR condition

You can use a UNION query to retrieve data from multiple companies: 

select *
from   dbtst120
where  empno > 100
union all
select *
from   dbtst120
where  empno < 100
and    _compnr = 100
