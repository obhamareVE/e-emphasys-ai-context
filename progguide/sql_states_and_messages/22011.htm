22011 - Substring error

An SQL statement contains a substring operator that has a negative length attribute.

Example
select * from dbtst120 where firstnme( 1; -1 ) = "C"

The length attribute has value -1, which is not allowed. This will result in the following error:

SqlState=22011, '', NativeError=309

It can be solved by assuring that the length attribute of a substring operator is positive. 

select * from dbtst120 where firstnme( 1; 1 ) = "C"

Note: The behavior on the example is different for different implementations of the SQL language.
