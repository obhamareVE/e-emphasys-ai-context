42I33 - REFERS TO predicate in HAVING clause

An SQL statement contains a REFERS TO predicate in a HAVING clause.

Example
select max(salary)
from   dbtst120, dbtst100
group  by edlevel
having workdept refers to dbtst100

REFERS TO predicates are only allowed in the WHERE clause. This query causes the error:

SQLState 42I33: REFERS TO predicate in HAVING clause
