42T08 - Incorrect operand types for operator

An SQL statement contains an expression with a binary operator and at least one of the operands of this expression does not have the correct type for this binary operator.

Example 1
select firstnme
from   dbtst120
where  firstnme = "CHRIS" + "TINE"

The binary operator '+' can not be applied to operands of type string. This query will cause the error:

SQLState 42T08: Incorrect operand types for operator '+'

Example 2
select firstnme
from   dbtst120
where  edlevel = 1 & 0

The binary operator '&' (concatenation) can only be applied to operands of type string.

SQLState 42T08: Incorrect operand types for operator '&'
