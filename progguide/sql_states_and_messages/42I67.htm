42I67 - Update on primary key column

An SQL UPDATE statement attempts to update a primary key column.

Example
update dbtst120
set empno = 0

Column empno is the primary key of table dbtst120. An SQL UPDATE statement cannot be used to update a primary key. This statement will cause the error:

SQLState 42I67: Update on primary key column 'empno'
