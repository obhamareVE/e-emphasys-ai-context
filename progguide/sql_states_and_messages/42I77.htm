42I77 - Invalid timestamp

The timestamp string is out of range or contains invalid values for the year, month, day, hour, minute or second fields.

Example
select timestamp '2006-11-28 11:61:59'
from   dbtst100

In this case the value for minutes is out of range. This query will cause the error:

SQLState 42I77: Timestamp string '2006-11-28 11:61:59' does not represent a valid timestamp

The valid range for timestamps is from '1970-01-01 00:00:00' up to '2038-01-19 03:14:07'. The fields have the following ranges:

YEAR 1970-2038
MONTH 1-12
DAY 1-31
HOUR 0-23
MINUTE 0-59
SECOND 0-59
