42T22 - Incompatible array depth for UNION column

An SQL statement contains a UNION operator and in the two queries that are combined by this UNION operator there are two corresponding select items that do not have the same depth.

Example
select phoneno
from   dbtst120
union  all
select firstnme
from   dbtst120

In this UNION the columns phoneno and firstnme are corresponding select items. Both columns are of type string, but column phoneno is an array column with depth 3 and column firstnme is not an array column.

SQLState 42T22: Incompatible array depth for UNION column 1
