22012 - Division by zero

An SQL statement contains an expression that divides a value expression by zero.

Example
select * from dbtst120 where edlevel = 38/0

The expression divides the constant 38 by 0, which is not allowed.

SqlState=22012, '', NativeError=306

It can be solved by assuring that the statement does not contain a division by zero. 

select * from dbtst120 where edlevel = 38/2
