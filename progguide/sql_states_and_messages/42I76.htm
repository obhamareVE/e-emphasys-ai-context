42I76 - Index hint specified on UNION query

A UNION query contains a HINT clause and this HINT clause contains an index hint.

Example
select mgrno
from   dbtst100
union  all
select respemp
from   dbtst160
hint   use index 1 on dbtst100

A HINT clause of a UNION applies to the UNION as a whole. You cannot specify an index hint in this HINT clause, because the index hint applies to only one of the queries of the UNION. This query will cause the error:

SQLState 42I76: Index hint specified on UNION query

You can use a SUBHINT clause to specify hints for a specific query in the UNION. For example, the above query can be corrected as follows: 

select  mgrno
from    dbtst100
subhint use index 1 on dbtst100
union   all
select  respemp
from    dbtst160
