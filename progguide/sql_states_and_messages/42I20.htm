42I20 - Aggregate function on combined column

An SQL statement contains an aggregate function that is applied on a combined column. This error will occur even if the combined column contains just one child column.

Example 1
select max(cmbb)
from   dbtst120
group  by edlevel

Column cmbb is a combined column that contains two child columns salary and bonus. The expression max(cmbb) will cause the error:

SQLState 42I20: Aggregate function on combined column 'cmbb'

Example 2
select avg(cmba)
from   dbtst120
group  by workdept

The column cmba is a combined column that contains only one child column: the column edlevel. The expression avg(cmba) will cause the error:

SQLState 42I20: Aggregate function on combined column 'cmba'
