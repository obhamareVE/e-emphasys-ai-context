42I48 - Cannot compare array values with non-array values or array values of different depth

An SQL statement contains a predicate in which two array columns with different depth are compared or in which an array column is compared to a non-array column, a parameter or a constant.

Example 1
select firstnme, lastname, phoneno
from   dbtst120, dbtst190
where  phoneno = strarr

Column phoneno is an array column with depth 3. Column strarr is an array column with depth 2. Because the depth of these columns is not the same, this query will cause the error:

SQLState 42I48: Cannot compare array values with non-array values or array values of different depth

Example 2
select firstnme, lastname, phoneno
from   dbtst120
where  phoneno(2) IN ( select phoneno
                       from   dbtst120
                       where  empno = 10 )

The array element phoneno(2) is compared to the entire phoneno array. This query will cause the error:

SQLState 42I48: Cannot compare array values with non-array values or array values of different depth

Example 3
select firstnme, lastname, phoneno
from   dbtst120
where  phoneno like "0342.*"

Because column phoneno is an array column this query will cause the error:

SQLState 42I48: Cannot compare array values with non-array values or array values of different depth
