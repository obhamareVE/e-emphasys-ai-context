42I73 - Unexpected select target

An SQL statement contains an explicit select target in a subquery or it contains an explicit select target in a query that is part of a UNION, but which is not the leftmost query of that UNION.

Example 1
select empno
from   dbtst120
where  empno in ( select mgrno :manager
                  from   dbtst100 )

The program variable manager is used as the explicit select target for the column mgrno of the subquery. Explicit select targets cannot be used in subqueries. This query will cause the error:

SQLState 42I73: Unexpected select target 'manager'

Example 2
select respemp
from   dbtst160
union  all
select mgrno :manager
from   dbtst100

Only the leftmost (or topmost) query of a UNION may contain explicit select targets. Here, the program variable manager is used as an explicit select target in the second query of the UNION. This will cause the error:

SQLState 42I73: Unexpected select target 'manager'
