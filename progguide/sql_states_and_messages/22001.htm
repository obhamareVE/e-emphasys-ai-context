22001 - String data right truncation

An assignment of a string or raw value would result in right truncation of the string.

Example
update dbtst120 set firstnme = 'xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
where empno = 10

Column is a string of length 20, and the value assigned to it is longer than 20 characters. This will result in the following error:

SqlState=22001, 'Value cannot be assigned to column dbtst120.firstnme', NativeError=205

It can be solved by assuring that the value assigned to the column is not longer than the column. 

update dbtst120 set firstnme = 'CHRISSIE'
where empno = 10
