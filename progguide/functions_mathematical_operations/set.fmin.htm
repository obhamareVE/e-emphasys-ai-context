set.fmin()
Syntax:

function void  set.fmin  (ref void variable [, string domain])

Description

This sets a specified string variable to its minimum value.

For standard variables, use set.min() instead.

For setting the maximum value, use set.fmax() and set.max().

Arguments
ref void variable

Reference argument to be set. This can be a single variable or an array.

[string domain]

Optional name of the domain of which the minimum value must be used.

If it is not supplied, then the runtime domain of the first argument is used.

If the first argument does not have a runtime domain, then a default domain is used which depends on the runtime type of the first argument.

 

Context

This function is implemented in the porting set and can be used in all script types.
