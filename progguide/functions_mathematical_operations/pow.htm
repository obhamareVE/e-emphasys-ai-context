pow()
Syntax:

function double pow  (double x, double y)

Description

This computes the value of x raised to the power of y. If x is zero, y must be positive. If x is negative, y must be an integer.

Arguments
double x
double y

 

Return values

The value of x to the power of y. If y is zero, the return value is always 1.0

Context

This function is implemented in the porting set and can be used in all script types.

Return values

The value of x to the power of y.

If y is zero, the return value is always 1.0.
