Mathematical operations overview

Use these functions to perform mathematical operations of various types – for example:

  • to perform logarithmic and square root calculations
  • to perform geometrical calculations (cosine, sine, tangent, and so on)
  • to perform rounding operations
  • to compare values and return the maximum or minimum
  • to set variables to their maximum or minimum values
  • to calculate the absolute value of numeric expressions
  • to convert floating point expressions to integer values
  • to compare the values of two doubles to determine if they are equal.
