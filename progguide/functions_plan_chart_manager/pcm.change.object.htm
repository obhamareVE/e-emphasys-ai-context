pcm.change.object()
Syntax:

function void pcm.change.object  (long plan_id, long object_id [, long flag] [, long value,])

Description

Deprecated. This function is only supported for Baan Windows and its usage is therefore deprecated.

Use this function to change the attributes of an object that was created by pcm.create.object().

Arguments
long plan_id

The ID of the chart to which the object belongs, as returned by pcm.create().

long object_id

The ID of the object, as returned by pcm.create.object().

[long flag]

Use these arguments to set new values for the object's attributes. For each attribute you specify, you must include the attribute type (for example, PcmMenuName or PcmTimescaleVisible), and the attribute value. For a list of the object attributes, see pcm.create.object().

[long value,]

Use these arguments to set new values for the object's attributes. For each attribute you specify, you must include the attribute type (for example, PcmMenuName or PcmTimescaleVisible), and the attribute value. For a list of the object attributes, see pcm.create.object().

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
