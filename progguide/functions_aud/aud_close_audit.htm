aud_close_audit()
Warning

The functions and macros listed below must not be used anymore. Using them in BaanERP 5.0c or higher releases may result in less reliable data. Know that there is a new set of functions that you can use for retrieval of audit information. Refer to Audit management overview

Syntax:

function long aud_close_audit  (long seqid)

Description

Discards the specified sequence identifier. seqid is the identifier returned by aud_open_audit() aud_open_audit when the particular sequence file was opened. When the last sequence file associated with an info file is closed, the info file is also closed.

Arguments
long seqid

 

Return values
0 success
-1 error (for example, could not close the sequence file or info file).

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
