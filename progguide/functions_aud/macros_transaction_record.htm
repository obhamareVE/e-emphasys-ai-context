Macros - transaction record
Warning

The functions and macros listed below must not be used anymore. Using them in BaanERP 5.0c or higher releases may result in less reliable data. Know that there is a new set of functions that you can use for retrieval of audit information. Refer to Audit management overview

Use these macros for mapping transaction records. All the macros take rec_buff as their input. This is filled by calling aud_read_audit_rec() or aud_read_next_audit_rec().

aud_get_rec_app_info() 
void aud_get_rec_app_info( string appinfo(4), 
                 string rec_buff() )

This reads the application-specific information of the record into appinfo.

aud_get_rec_op_type() 
string aud_get_rec_op_type( string rec_buff() )

This returns the type of action of the record. Possible values are:

I insert record
D delete record
U update record
L clear table
R drop table
C create table

 
aud_get_rec_seqno() 
long aud_get_rec_seqno( string rec_buff() )

This retrieves the sequence number of the record within the transaction.
