Chart manager glossary
axis One of two or more reference lines, positioned at right angles to each other, against which data values are plotted. In mathematics, X, Y, and Z axes are used. The Chart Manager uses four axes:

category axis as x axis

data axis as y axis

series axis as z axis

data axis II as second y axis

You use data axis II when you want to draw two series to different scales in the same graph.

axis label This indicates the meaning or value of a major division on the axis.
back plane The plane behind the graph (that is, the background).
bar chart A graph in which a vertical or horizontal bar represents each data value. Bars representing the same series have the same color, pattern, and width. The higher the bar, the higher the value at that point. The bar width and thickness (3D only) can be set per series. Bars representing different series can overlap wholly or partially.
bottom plane The plane below the graph.
category Measuring points are divided into categories. For example, when plotting the quarterly expenditures in a particular year, each quarter is regarded as an information category. Category values are measured on the x-axis. In pie charts, each segment can be regarded as a category, depending on the way the data is sorted.
chart A chart is a graphical presentation of information. A chart displays a pie and/or graph including title, legend, and footnotes.
Chart Manager A tool within the Infor ERP Enterprise application, which presents data in graphs and pie charts.
Chart Manager application A collection of charts, user option sets and window preferences. The Chart Manager needs this information for its operation.
chart type The chart type specifies the attributes of a chart – for example, the graph type, the thickness of lines and bars, colors, and fonts. These settings are data- independent.
data The information presented in the chart – for example, the domains used, the data point values, and so on.
data label The name or the value of a data point.
data number The sequence number of a data point within a category.
data point A point in the chart representing a value, and displayed, for example, as a symbol, a bar, a pie segment, or a kink in a line.
division The axes of a graph are divided into a number of major and minor divisions. These divisions are marked by ticks. Grids and labels are assigned by major division.
division labels Special labels at major divisions. These replace the values from the domain associated with the axis.
domain A range with a start value, end value and step size. These indicate which values are legal. There are data, category, and series domains.
domain type In the Chart Manager there are three domain types:

integer the domain consists of a series of integers

real the domain consists of both fractions and integers

enumerated set the domain consists of a set of labels

exploded A detached pie segment. You can use this option to draw attention to one of the segments. The distance from the segment to the pie center can be specified as a percentage of the pie's radius.
footnote Information relating to a data point or projection point. Footnotes are placed in a separate box in the chart.
footnote label The text placed in a footnote box.
footnote number The number or the series of asterisks placed at the data point or projection point and referring to the same number or series of asterisks in the footnote box.
graph A graphical presentation of information drawn within a set of axes.
grid A pattern of lines at right angles to each other and the axis. They connect the values in the graph with the points on the vertical and horizontal axes. Grid lines are drawn at major divisions only.
hilo (high-low) graph A hilo graph contains multiple data points per category, each data point having its own meaning. For each category there may, for example, be a data point representing the maximum value and one representing the minimum value. This type of graph is very suitable, for example, for monitoring stock market developments.
hilo selects Symbols in a hilo graph indicating where in a category a particular data point is located.
histogram A histogram consists of a set of stacked blocks, each block representing a particular data value.
intersection point The value on the data axis where category axis and data axis intersect.
legend An explanation of the chart. The legend contains a sample of each series (bar, line, pie, etc.) in the chart with a description of the series.
line In a line graph, the data points for each series are connected by a line. There are four types of lines: curve, linear, stepped and wire. Line graphs are mainly used to show trends.
linear A line graph in which data points are joined by straight lines.
logarithmic A scaling in which the axis is divided into equal sections that do not represent equal steps in a scale of values. Instead the divisions increase by a specified factor. Logarithmic scales are useful when there is a wide range of values.
option set A number of possibilities or limitations per user or group.
owner The creator of the chart. Users can modify only their own charts.
percent label The proportion of the total pie value represented by a pie segment.
pie Circular diagram divided into segments showing the quantitative relationship between different series or categories. The pie represents the total value. Pies are the only graphs that are not axis-oriented.
pie segment A part of the pie. Each segment represents a percentage of the total pie.
projection A way of presenting a graph in which the graph is split up in different parts referred to as projection areas. Each projection area has its own attributes. A series can be shown by a different graphical means in each projection area. The use of several projection areas is a convenient means to distinguish between measured and expected (projected) values in a series. Projection areas are separated by a projection line, which is placed at right angles to the category axis.
projection point The point on the category axis where the projection line and the category axis intersect.
scale The size of the graph or pie as a percentage of the box size.
second data axis The fourth axis of the Chart Manager's coordinate system. It has the same dimensions as the data axis (y-axis). The second data axis is always used in combination with the first data axis when two (or more) different series must be drawn to different scales in the one chart.
second data domain The second data domain is used for the second data axis.
series A set of related data points that are linked graphically. In a line graph, for instance, they are connected by a line. This line indicates which data points constitute one series.
set A group of labels.
side plane The plane beside the graph.
stepped A line graph in which the line consists of horizontal and vertical sections. Stepped lines draw attention to changes in the level of data values.
symbol A sign indicating where a data point is located.
ticks Signs indicating the divisions on the axis. Ticks at major divisions are larger than those at minor divisions.
title In the Chart Manager there are four types of titles:
  • The description of the Chart Manager application. This appears at the top of the form together with the package and module codes.
  • Main title (placed in the title box).
  • Subtitle (placed in the title box).
  • Axis title. This is the description of an axis and is the same as the domain label.
unit label The description of the unit (measure or quantity) in which categories, data or series are expressed. The label is placed beside the relevant axis.
wire A line graph in which the data points of different series are connected by lines, the result being a kind of mountain landscape. This is possible in 3-D graph only.
