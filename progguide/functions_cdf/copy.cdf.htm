copy.cdf()
Syntax:

function long copy.cdf  (long src.table.id, long dest.table.id)

Description

This function facilitates thet application in the task of carrying forward the information in the CDFs through the business process. The function will copy CDFs from a source record buffer to a destnation record buffer if (1) the destination table has a CDF with the same name and (2) the value in the destination record buffer is empty.

Example:

copy.cdf( ttdsls101, ttdsls401 )

Example:

quotation.line.table = db.bind("tdsls101", my.quot.line.buf ) order.line.table = db.bind("tdsls401, my.order.line.buf ) . . . copy.cdf(quotation.line.table, order.line.table)

Arguments
long src.table.id

source table
long dest.table.id

destination table

 

Return values
0 Success.
<>0 Error.

 

Context

This function is implemented in the porting set and can be used in all script types.
