db.record.changed()
Syntax:

function boolean db.record.changed  (long table_id)

Description

Tests whether the current value of the record buffer belonging to the given table id has changed since the last call of db.store.record().

First, a db.columns.to.record() is done in order to copy the table field values to the record buffer. Then this record buffer is compared to the one stored with db.store.record().

Note

Note that, because of performance reasons, db.record.changed() is implemented as a macro. This is the actual implementation: 

#define db.record.changed(tbl)
^       (db.columns.to.record(tbl) <> 0 or changed(rcd.##tbl, KEEP.CHANGED.FLAG.RAISED))

Arguments
long table_id

The table ID in the format tppmmmxxx. E.g. twhinh200

 

Return values
true Record was changed since the last call to db.store.record()
false Record was not changed

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Note

If db.store.record() has not been called before, this function returns TRUE.
