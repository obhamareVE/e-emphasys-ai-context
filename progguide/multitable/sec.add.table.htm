sec.add.table()
Syntax:

function long sec.add.table  (const string tablename(),  ...)

Description

Add the passed table as a secondary table to the current session. This function call is only allowed from the before.program section.

Arguments
const string tablename()

The table name to be added.
...

(optional) pairs of parameters of type string which defines the mapping of maintable fields on secondary table keyfields.

First string of pair: maintable field

Second string of pair: secondary table field

 

Return values

The id of the secondary table. This id must be used in subsequent calls to the functions sec.add.set(), sec.get.update.status(), sec.mark.delete() and sec.record.exists().

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types. This function is available from TIV level 1075.
