stop.synchronized.child()
Syntax:

function long  stop.synchronized.child  (long child.id)

Description

This ends a child session.

Arguments
long child.id

The unique identifier for the child session to be stopped, as returned by start.synchronized.child() or start.synchronized.child.with().

 

Return values

0         success

<0         error

Context

This function is implemented in the 4GL Engine and can be used in all script types.
