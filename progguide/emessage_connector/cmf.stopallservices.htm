cmf.stopAllServices()
Syntax:

#include <bic_cmf>

function void cmf.stopallservices  ()

Description

Stops all Infor ERP Enterprise eMessage Connector services started by cmf.sendToPerson().

Return values

None.

Context

This function is implemented in the 4GL Engine and can be used in all script types.
