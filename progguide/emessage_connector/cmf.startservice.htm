cmf.startService()
Syntax:

#include <bic_cmf>

function long cmf.startservice  (long service(15), long mode [, string address(13)])

Description

Starts the Infor ERP Enterprise eMessage Connector service

Arguments
long service(15)

The service offered by a provider, for example, Fax or Email.

long mode
[string address(13)]

 

Return values
0 Success.
-1 Service not present.
-2 Service could not be started.
-3 Service not enabled.

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
