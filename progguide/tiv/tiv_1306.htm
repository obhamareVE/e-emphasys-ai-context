TIV 1306

With TIV 1306 the behavior of ncrs selection dialogs has been changed:

If records have been selected on the parent session, the selection group related to the main table of the parent session will not be visible. If no records are selected on the parent session or the session has been started directly, all the selection groups are visible. The selection choice group will never be visible anymore. If no fields are left on the dialog, the default command will be triggered and the session will be ended. In the init.form section it will be possible to trigger another command than the default command if needed.

    • ncrs.is.session.empty()
