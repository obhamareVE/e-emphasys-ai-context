path.is.absolute()
Syntax:

function boolean  path.is.absolute  (string path_name)

Description

This checks if path_name is an absolute path.

Arguments
string path_name

Path name.

 

Return values
false path_name is not absolute.
true path_name is absolute.

 

Context

This function is implemented in the porting set and can be used in all script types.
