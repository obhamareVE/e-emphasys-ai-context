seq.unlock()
Syntax:

function long  seq.unlock  (long mode, long offset, long size, long fp)

Description

This removes a lock set by seq.lock().

Arguments
long mode

The type of lock to remove:

SEQ_R_LCK read lock
SEQ_W_LCK write lock
SEQ_F_R_LCK file lock (read)
SEQ_F_W_LCK file lock (write)

 
long offset

To remove a lock from a segment of the file, use these arguments to specify the offset of the segment from the beginning of the file and the length of the segment.

To remove a lock that extends to the end of the file, set size to 0.

To remove a lock from the entire file, set both arguments to 0.

long size

To remove a lock from a segment of the file, use these arguments to specify the offset of the segment from the beginning of the file and the length of the segment.

To remove a lock that extends to the end of the file, set size to 0.

To remove a lock from the entire file, set both arguments to 0.

long fp

The file pointer returned by seq.open() when the file was opened.

 

Return values
>=0 Success.
-1 Error; probably fp not connected.
< -1 Error; Negative error value.

 

Context

This function is implemented in the porting set and can be used in all script types.

Notes

All locks set on a file by a particular process are removed automatically when the file descriptor for the file is closed by the process (see seq.close()) or when the process itself terminates.
