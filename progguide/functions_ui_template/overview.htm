UI Template
Overview

The 4GL engine supports automatic disabling of fields and commands. Instead of using functions like disable.fields() and disable.fields(), a set of so-called hooks can be used that describe under which conditions a field or command is enabled. The 4GL engine will call these hooks at the right moment and based on the outcome, the corresponding field or command will be disabled or enabled.

The following UI template topics are supported by the 4GL engine:

  • field.is.readonly()
  • standard.command.is.allowed()
  • form.command.is.allowed()
Note

In order to enable automatic disabling of fields and commands, you must #include <bic_4gl2> in your UI script.
