pipe.write()
Syntax:

function long  pipe.write  (string buffer, long nr_of_bytes, long pipe_id)

Description

This appends a specified number of bytes to the specified pipe.

Arguments
string buffer

This stores the characters that must be written to the pipe.

long nr_of_bytes

This specifies the maximum number of bytes to be written to the pipe. The function stops writing to the pipe when it has written this number of bytes or when it reaches the end of the file, whichever comes first.

long pipe_id

The pipe ID, as returned by pipe.open().

 

Return values

>0       the number of bytes written

0       end-of-file

-1       error; most probably no connected stream on pipe_id

pipe_id

Context

This function is implemented in the porting set and can be used in all script types.
