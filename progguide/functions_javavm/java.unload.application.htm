java.unload.application
Syntax:

function long java.unload.application  (long handle)

Description

Unloads an application loaded by java.load.application.

This will free the resources, stop the JVM (if this was the last application using it) and invalidate the handle.

Arguments
long handle

The handle of the application to unload, previously obtained by a successful call to java.load.application.

 

Context

This function is implemented in the porting set and can be used in all script types. This function is available from TIV level 2020.

Return codes
0 Success.
<0 Problems occurred. Additional details will be available in the message log.

 

For an example, see java.load.application
