ims.getc$()
Syntax:

function string ims.getc$  (long bytearray)

Description

Returns the next character from the byte array.

Arguments
long bytearray

The byte array identifier that is returned by ims.openvba or ims.openfba.

 

Return values

If an empty string is returned, either the end of the string has been reached or an error has occurred.

Context

This function is implemented in the porting set and can be used in all script types.
