Byte arrays overview

There are two types of byte arrays:

  • Fixed Byte Arrays (FBA)
  • Variable Byte Arrays (VBA)
Fixed Byte Arrays

This type of array has a fixed size, and memory must be allocated by the application developer. See ims.openfba()

Variable Byte Arrays

This type of array has a variable size that can grow as the program script runs. The growth is in a fixed number of bytes that can be set by the application developer. The memory allocation is done by the Bshell. See ims.openvba()
