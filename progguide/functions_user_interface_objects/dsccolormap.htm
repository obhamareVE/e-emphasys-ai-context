DsCcolormap
Description

A color map defines an array of colors that can be used by a DsCpixmap object. The DsNdata attribute of a pixmap object contains an array of indexes into the color table of a DsCcolormap object.

Events

A DsCcolormap object does not generate events.

Attributes

DsNcolorTable

(long array)

 [CSG] An array of rgb values.

DsNobjectType

(long)

 [G] The object type.

DsNparent

(long)

 [G] The ID of the parent object.

DsNtemplate

(long)

 [CS] The ID of a DsCtemplate that defines a set of attributes to be applied to the object.
