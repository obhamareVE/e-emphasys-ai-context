map.object()
Syntax:

function void  map.object  (long object_id)

Description

Deprecated. This function is only supported for Baan Windows and its usage is therefore deprecated.

This makes the specified object visible, without changing the order of the objects on the workspace.

Arguments
long object_id

 

Context

This function is implemented in the porting set and can be used in all script types.
