gbf.get.selected.nr()
Syntax:

function long  gbf.get.selected.nr  ()

Description

This function returns the current number of selected objects. It can be used to obtain the maximum allowed index for gbf.get.selected().

Return values
> 0 Successful completion, the actual number of selected objects is returned
= 0 Successful completion, but currently no objects are selected
GBF.ILL.STATE GBF is not in the right state to deal with this function

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
