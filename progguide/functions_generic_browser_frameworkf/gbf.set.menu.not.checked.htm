gbf.set.menu.not.checked()
Syntax:

function long gbf.set.menu.not.checked  ()

Description

This function is actual a define and is defined as:

gbf.set.menu.not.checked(item.id) gbf.set.menu.state(0, item.id, false, true)

 

This function can place a ticmark on a menu item.

The menu.id is the return value of the function gbf.set.menu.head() when that function was successfully completed.

See gbf.set.menu.state() for more details.

Arguments:
item.id The return value of the function gbf.set.menu.item() when that function was successful completed.

 

Return values
0 Successful completion
GBF.ILL.MENU.ID Unknown menu.id or item.id specified
GBF.ILL.STATE GBF is not in the right state to deal with this function

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.

gbf.set.menu.checked(), gbf.set.menu.disabled(),gbf.set.menu.enabled(), gbf.set.menu.not.checked(), gbf.set.menu.not.radio(),gbf.set.menu.radio()
