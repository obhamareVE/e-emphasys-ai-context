gbf.set.selected()
Syntax:

function void gbf.set.selected  (long obj.id)

Description

This function can be used to set the selected object in a GBF

Arguments
long obj.id

The object identification

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
