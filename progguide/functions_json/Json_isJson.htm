Test JSon value
Syntax:

function boolean Json.isJson  (long json)

Description

Tests whether parameter 'json' is a JSON value.

Arguments
long json

json is a JSON value.

 

Return values

True if parameter 'json' is a JSON value, else false.

Context

This function is implemented in the 4GL Engine and can be used in all script types.
