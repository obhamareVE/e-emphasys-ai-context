Add a null object to a JSon object
Syntax:

function long Json.addNull  (long json)

Description

Convenience function for adding a JSON null value to a JSON array. The function is a shorthand for: Json.add(json, Json.newNull())

Arguments
long json

json is a JSON array.

 

Return values

Optional, the new JSON null value which has been added to the JSON array.

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Preconditions
  • Parameter 'json' is a JSON value of type JSON_TYPE_ARRAY.
