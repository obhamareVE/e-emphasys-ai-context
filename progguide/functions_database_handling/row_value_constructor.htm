Row value constructor

The row value constructor constructs a row value from a value or a list of values.

Syntax
<row value constructor>
    ::= Column reference
      | Value expression
      | { Value expression [ , Value expression ]... }
Examples
3.14

{ salary * 2.20371 }

{ 'Hello ' & 'World!' }

{ 'tt', 'adv', '101' }

{ firstnme, lastname }

{ AVG( salary ), AVG( bonus ) }
