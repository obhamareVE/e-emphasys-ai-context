Enumerate constant

The enumerate constant specifies the value of an enumeration type, as defined in the data dictionary.

Syntax
<enumerate constant>
    ::= Identifier
Semantics

The type of an enumerate constant is integer. The value is the integer value that according to the data dictionary corresponds with the identifier.

Examples

The following specifies the integer value 1.

dbsex.male
