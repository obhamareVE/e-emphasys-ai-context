disable.db.dll()
Syntax:

function void disable.db.dll  ()

Description

This disables the call of the db hooks in the db dll. (db.before.insert, db.before.update, db.before.delete, db.after.insert, db.after.update, db.after.delete)

Context

This function is implemented in the 4GL Engine and can be used in all script types.
