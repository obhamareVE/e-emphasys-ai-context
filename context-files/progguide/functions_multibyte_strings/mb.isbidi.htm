mb.isbidi()
Syntax:

function boolean  mb.isbidi  ()

Description

This function checks whether the current environment is a bidirectional one.

Return values

TRUE        current environment is bidirectional

FALSE      current environment is not bidirectional

Context

This function is implemented in the porting set and can be used in all script types.
