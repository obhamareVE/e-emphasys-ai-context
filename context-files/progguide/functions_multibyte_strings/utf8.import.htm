utf8.import()
Syntax:

function long  utf8.import  (ref string target$, const string source$, long option_mask)

Description

This function converts a string from Unicode (encoded according to a UTF-8 Encoding encoding) to the TSS Encoding character set.

Arguments
ref string target$

The target string. This string will receive the TSS encoded characters. This can be a maximum of 4096 bytes.

const string source$

The source string. It is assumed to be in a UTF-8 encoding.
long option_mask

This specifies the format of the source string.

UTF8_STD_MODE Assume that is a standard UTF-8 string.
UTF8_JAVA_MODE Assume that is a Java modified UTF-8 string.

 

 

Return values

>= 0       the number of converted bytes

-1       the target string was too small to contain the converted string

-2       an incomplete character was found at the end of the string

-3       character could not be converted.

Context

This function is implemented in the porting set and can be used in all script types.
