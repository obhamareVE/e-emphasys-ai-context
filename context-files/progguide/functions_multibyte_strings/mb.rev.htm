mb.rev$()
Syntax:

function string  mb.rev$  (string string_expr [, long flags])

Description

This function reverses the order of the characters in a specified string.

Arguments
string string_expr

The source string.
[long flags]

By default, this function reverses the string, whether or not the current user locale is bidirectional. Use these optional flags to override this default situation:

TSS_REVERSE                    reverse string only if it contains bidirectional

characters and the current locale is bidirectional

TSS_FORCE_REVERSE     reverse string only if it contains bidirectional

characters

 

Return values

The reversed string.

Context

This function is implemented in the porting set and can be used in all script types.

Notes

The order of ASCII characters does not change. Code features are treated as separators and their position does not change. For example:

h1, h2, and h3 are bidirectional characters

h1 h2 h3 A B C cf(0) D E F

 

Reversed string: Hebrew/Arabic

D E F cf(0) A B C h3 h2 h1

 

Examples
x = mb.rev$( y  )                       | reverses always
x = mb.rev$( y, TSS_REVERSE )
                                        | reverses when current locale is
                                        | bidirectional and y contains bidirectional characters
if mb.hasbidi( y ) then
                x = mb.rev$( y )        | reverses when y contains bidirectional characters
endif
