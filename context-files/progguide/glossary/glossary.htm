Glossary
4GL engine

The program that provides default functionality for a session to prevent application programmers from having to develop a session from scratch. The 4GL engine, formely called standard program (STP), is used because essentially sessions are alike. In addition to provide default functionality, the 4GL engine also provides a mechanism to change the 4GL engine's default behaviour, and to program dedicated functionality for a specific session. Each time a session is started, a separate 4GL engine instance is activated to handle the session.

ARTM

Acronym for Application Response Time Measurement.

BSE

Acronym for Baan Software Environment.

DFE

Acronym for Dynamic Form Editor. With this editor the fields on a form are defined. It doesn't used exact positioning, but the developer set the relative position of a field in relation to other fields.

pid

Acronym for Process ID (?).

SCM

Acronym for Software Configuration Management.
