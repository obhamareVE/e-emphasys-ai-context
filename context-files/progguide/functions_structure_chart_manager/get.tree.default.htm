get.tree.default()
Syntax:

function void  get.tree.default  ()

Description

Deprecated. This function is only supported for Baan Windows and its usage is therefore deprecated.

When the client receives a bucket message that cannot be retrieved with get.tree.node.press(), get.tree.node.dpress() or get.tree.push.button(), call this function to remove the bucket message from the internal stack.

Context

This function is implemented in the 4GL Tools and can be used in 4GL script types.
