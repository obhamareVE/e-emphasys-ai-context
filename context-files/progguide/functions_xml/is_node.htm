Check a long is a valid XML node
Syntax:

function boolean xmlIsNode  (long node)

Description

Check whether the value supplied is a valid XML node.

Arguments
long node

node is the node to be checked whether it is (still) a valid XML node.

 

Return values
true node is a valid XML node
false node is not a valid XML node

 

Context

This function is implemented in the porting set and can be used in all script types.
