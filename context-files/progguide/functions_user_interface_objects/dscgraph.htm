DsCgraph
Description

A graph is a rectangular child window that displays a graph. The DsCgraph object is not part of the standard Infor ERP Enterprise Server object library. It is a third-party OCX control. It can be controlled from 3GL scripts in the same way as standard objects. Several graphs can be displayed simultaneously within the same main window. A wide range of graph types is supported.

Events

A DsCgraph object can generate the following events:

EVTBUTTONPRESS

EVTBUTTONRELEASE

The parameters of these events are as described in Event array parameters, with the following exceptions:

evt.button.x           The graph set selected.

evt.button.y           The graph point selected.

evt.button.rootx     The x-coordinate (Graphic Server units).

evt.button.rooty     The y-coordinate (Graphic Server units).

evt.button.state     0 when a graph set/point is selected

(Shift and Control states are not supported).

Attributes

DsNbackgroundColor

(long)

 [CSG]

The background color for the graphic object specified by the  DsNbackgroundUse attribute. Possible values are:

DSBLACK                  Black

DSBLUE                    Blue

DSGREEN                 Green

DSCYAN                    Cyan

DSRED                      Red

DSMAGENTA             Magenta

DSBROWN                Brown

DSLGRAY                  Light gray (default)

DSDGRAY                  Dark gray

DSLBLUE                   Light blue

DSLGREEN               Light green

DSLCYAN                  Light cyan

DSLRED                   Light red

DSLMAGENTA          Light magenta

DSYELLOW               Yellow

DSWHITE                   White

DSAUTOCOLOR       Automatic black or white (whichever

provides the greatest contrast)

DsNbackgroundStyle

(long)

 [CSG]

The background style for the graphic object specified by the DsNbackgroundUse attribute. Possible values are:

DSBGSTYLENO                    No background (default).

DSBGSTYLESHADOW         Drop shadow.

DSBGSTYLERAISED            Sculptured raised.

DSBGSTYLELOWERED      Sculptured lowered.

DsNbackgroundUse

(long)

 [CSG]

Specifies the graphic object to which the DsNbackgroundStyle and DsNbackgroundColor attributes apply. Possible values are:

DSBGENTIRE                        Entire graph window (default).

DSBGGRAPHAXES               Graph and axes.

DSBGGRAPHTITLE              Top title.

DSBGLEFTTITLE                  Left title.

DSBGRIGHTTITLE                Right title.

DSBGBOTTOMTITLE           Bottom title.

DSBGLEGEND                      Legend.

DsNbottomTitle

(string)

 [CSG] The text of the graph title positioned at the bottom of the graph window. This can be up to 80 characters long, and can include new-line characters. Use DsNbackgroundStyle and DsNbackgroundUse to change the style of the title.

DsNdrawMode

(long)

 [CS]

The drawing mode for the graph control. Possible values are:

DSDRAWNOACTION            Graph not drawn – reset to

DSDRAWDEFAULT to display the graph.

DSDRAWCLEAR                   Graph not drawn but background set to

color specified by DsNbackground.

DSDRAWDEFAULT              Redraws the graph immediately

(default).

DSDRAWBLIT                       Graph built in background, then

displayed all at once.

DSDRAWCOPY                    Copies the graph image as a bitmap to

the clipboard.

DSDRAWROTATE                Redraws the graph itself without

recalculating scales and redrawing titles

and legends (True3D graphs only).

DSDRAWUPDATE                Updates a time series graph with newly

added points.

DsNelevation

(long)

 [CSG] The vertical viewing angle for true3D graphs. The value can be any angle between -60 degrees and 90 degrees.

DsNextraData

(long array)

 [CS]

A two-dimensional array that holds additional data for a graph. You fill the array in the same way as you fill the DsNgraphData array. When referencing a single data point, use DsNthisSet and DsNthisPoint to index the data point. Possible values are:

Pie charts

DSEDNORMAL          Not exploded (default).

DSEDEXTRA             Exploded.

Line, Lin/log, Log/lin, Log/log, Polar, Scatter (2D), Tape

DSEDNORMAL          Point shown (default).

DSEDEXTRA             Point not shown.

Time series

DSEDNORMAL          No statistical lines (default).

DSEDEXTRA             Mean line.

DSEDSTDDEV          Standard deviation line.

DSEDEXTRA and DSEDSTDDEV can be combined.

DsNforegroundColor

(long)

 [CSG] The foreground color for the graphic object specified by the DsNforegroundUse attribute. See DsNbackgroundColor for the list of possible values. The default is DSAUTOCOLOR.

DsNforegroundUse

(long)

 [CSG]

Specifies the graphic object to which the DsNforegroundColor attribute applies. Possible values are:

DSFGRAPHTITLE                 Top title (default)

DSFGLEFTTITLE       Left title

DSFGRIGHTTITLE                Right title

DSFGBOTTOMTITLE            Bottom title

DSFGLABELS                       Labels

DSFGLEGEND                      Legend

DSFGAXES                            Axes

DSFGGRIDS                          Gridlines

DSFGMEAN                           Mean line

DSFGMINMAX                        Min and max lines

DSFGSTDDEV                      Standard deviation lines

DSFGBESTFIT                      Best-fit line

DSFGCURVE             Fitted curve

DSFGDATALABELS              Data labels

DsNgraphData

(long array)

 [CS]

A two-dimensional array that holds the actual data values for a graph. The array is indexed by DsNthisSet and DsNthisPoint.

For a graph with two data sets and three data points per set, the array data is filled as follows:

set1_pt1 + set1_pt2 + set1_pt3 + set2_pt1 + set2_pt2 + set2_pt3

DsNlabels

(long)

 [CSG]

Enables/disables graph labels. Possible values are:

DSDISABLE               No labels.

DSENABLEBOTH      Both X and Y labels (default).

DSENABLEX              X labels only.

DSENABLEY              Y labels only.

DsNlabelEvery

(long)

 [CSG] The frequency of X axis labels. You can specify any frequency from 1 (every data point is labeled) to 1000 (every 1000th data point is labeled, starting with the first point).

DsNlabelStyle

(long)

 [CSG]

The orientation of X and Z axes labels. For horizontal bar graphs and gantt graphs, the label style applies to the Y axis rather than the X axis. For other graph types, Y axis labels are always horizontal. Possible values are:

DSLABELHORIZONTAL        Horizontal (default)

DSLABELVERTICAL             Vertical

DsNlabelText

(void)

 [CS]

A one-dimensional array that holds the text for graph labels.

For graph types with X and Y axes, the labels apply to data points on the independent axis (normally the X axis). For pie charts, the labels apply to each pie slice. The default labels are the numbers of the data points.

The array strings are filled from left to right, separated by null characters. Each string can be up to 80 characters long. To label one particular data point, you can index DsNlabelText using DsNthisPoint.

Use DsNyLabelText and DsNzLabelText to specify label text for the Y and Z axes of a graph.

DsNleftTitle

(string)

 [CSG]

The text of the graph title positioned on the left side of the graph window. This can be up to 80 characters long, and can include new-line characters.

Use DsNbackgroundStyle and DsNbackgroundUse to change the style of the title.

DsNleftTitleStyle

(long)

 [CSG]

The orientation of the graph title specified by DsNleftTitle. Possible values are:

DSTITLENORMAL     Horizontal (default).

DSTITLEUP                Vertical (read from bottom to top).

DSTITLEDOWN         Vertical (read from top to bottom).

DsNlegendPos

DSLEGENDTOP                               Top

DSLEGENDTOPLEFT                      Top left

DSLEGENDLEFT                              Left

DSLEGENDBOTTOMLEFT              Bottom left

DSLEGENDBOTTOM                       Bottom

DSLEGENDBOTTOMRIGHT            Bottom right

DSLEGENDNO                                 No legend (default)

DsNlegendSize

(long)

 [CSG] Specifies the size of the gap between lines in the legend as a percentage of the maximum possible gap. This latter varies depending on the size of the graph window and the number of items in the legend. You can specify any value from 0% to 100% (default).

DsNlegendStyle

(long)

 [CSG]

Specifies the color for the legend text. Possible values are:

DSLEGENDFG                      Legend text set by DsNforegroundColor.

DSLEGENDCOLOR              Legend text in same colors as legend

DsNlegendText

(long)

 [CS]

A one-dimensional array that holds text for the graph legend.

For graph based on a single data set (pie, bubble, and single-set bar graphs), the legends apply to the individual data points. For other graph types, the legends apply to the data sets.

The array strings are filled from left to right, separated by null characters. Each string can be up to 80 characters long.

DsNlineStats

(long)

 [CSG]

Enables/disables statistical lines and curves. Both lines and curves are available for line, 2D scatter, high-low-close, open-high-low-close, and box-whisker parametric graphs. Only curves are available for candlestick graphs. Possible values are:

DSSTATNOLINES     No statistical lines (default).

DSSTATMEAN           Mean (average) line.

DSSTATMINMAX        Min/max line.

DSSTATSTDDEV      Standard deviation line.

DSSTATBESTFIT      Best-fit line.

DSSTATCURVE        Curve line.

You can combine all the above values.

DsNnumPoints

(long)

 [CSG]

The number of points in the data sets of a graph. This can be any value between 1 and 6000, provided that the total number of points in a graph does not exceed 128000. That is:

DsNnumSets * DsNnumPoints <= 128000

DsNnumSets

(long)

 [CSG]

The number of data sets in the graph. For most graph types, DsNnumSets can be any value between 1 and 6000. The exceptions are:

Pie (2D/3D)     1 set

Bubble             2 sets

Gantt               >=2 sets

Surface           >=2 sets

HLC                 3 sets

OHLC              4 sets

Candlestick     4 sets

Box-whisker    7 sets for parametric data (can be higher for raw data)

The total number of sets in any graph is limited to 128000. That is:

DsNnumSets * DsNnumPoints <= 128000

 [CSG] The degree of perspective foreshortening for true3D graphs that use the perspective projection method. This represents the perceived distance from which the graph is viewed. The value must lie in the range 1 to 100 (default is 50).

Use DsNbackgroundStyle and DsNbackgroundUse to change the style of the title.

DsNrightTitleStyle

(long)

 [CSG]

The orientation of the graph title specified by DsNrightTitle. Possible values are:

DSTITLENORMAL     Horizontal (default).

DSTITLEUP                Vertical (read from bottom to top).

DSTITLEDOWN         Vertical (read from top to bottom).

DsNrotation

(long)

 [CSG] The horizontal viewing angle for true3D graphs. The value can be any angle between -180 degrees and 180 degrees.

DsNsetState

(long)

 [CS] The state of the object. See DsCmwindow.

DsNtemplate

(long)

 [CS] The ID of a DsCtemplate that defines a set of attributes to be applied to the object.

DsNthisPoint

(long)

 [CS]

This indexes array-based attributes by data point number. In combination with the DsNthisSet attribute, it indexes individual data points in a data set for direct access.

The following example modifies a pie graph, exploding points 3 and 5, and imploding point 4:

long   extra_array(10)

extra_array(1) = DSEDEXTRA  |* Explode point

extra_array(2) = DSEDNORMAL |* Return point

|* to normal

extra_array(3) = DSEDEXTRA  |* Explode point

change.object( pie_graph_object,

DsNthisSet,    1,    |* Pie graph has only 1 set

DsNthisPoint,  3,    |* Start with point 3

DsNextraData, extra_array, 3  |* Do for 3 points

)

DsNthisSet

(long)

 [CS]

This indexes array-based attributes by data set number. In combination with the DsNthisPoint attribute, it indexes individual data points for direct access. DsNthisSet can be used to index the following attributes: DsNextraData, DsNgraphData, DsNlegendText, and DsNzLabelText.

The following example changes the value of one data point, using DsNthisSet and DsNthisPoint to index the data point:

long data_array(100)

data_array(1) = 50

change.object(graph_object,

DsNthisSet,     3,

DsNthisPoint,   6,

DsNgraphData,   data_array, 1

)

DsNtickFrequency

(long)

 [CSG] The frequency of ticks on the X axis (or on the Y axis of horizontal bar graphs and gantt graphs). You can specify any frequency from 1 (every data point is labeled) to 1000 (every 1000th data point is labeled, starting with the first point).

DsNtickStyle

(long)

 [CSG]

Enables/disables tick marks on the graph axes. Possible values are:

DSDISABLE               No ticks

DSENABLEBOTH      Both X and Y ticks (default)

DSENABLEX              X ticks only

DSENABLEY              Y ticks only

DsNtrue3D

(long)

 [CSG]

Enables/disables true3D graphing and sets the projection method. Possible values are:

DSTRUE3DOFF                                No 3D (except for scatter and

surface graphs).

DSTRUE3DPERSPECTIVE3D         Perspective projection (see

DsNperspective).

DSTRUE3DISOMETRIC3D  Isometric projection.

DSTRUE3DOFF is the default when true3D is not available for the graph type. DSTRUE3DPERSPECTIVE is the default when true3D is available for the graph type.

DsNwidth

(long)

 [CSG] The width of the object, in pixels.

DsNx

(long)

 [CSG] The x-coordinate of the object's outer left edge, in pixels, relative to the inner left edge of its parent.

DsNy

(long)

 [CSG] The y-coordinate of the object's outer top edge, in pixels, relative to the inner top edge of its parent.

DsNyLabelText

(void)

 [CS] A one-dimensional array that holds the text for Y axis labels. The default labels indicate the numeric scaling of the axis. The array strings are filled from left to right, separated by null characters. Each string can be up to 80 characters long.

DsNzLabelText

(void)

 [CS]

A one-dimensional array that holds the text for Z axis labels in true3D graphs.

The default labels indicate the numeric scaling of the axis. However, no default labels are supplied for true 3D bar graphs, tape graphs, area graphs, or surface graphs. In these cases, you must set DsNzLabelText if you want to display z axis labels.

The array strings are filled from left to right, separated by null characters. Each string can be up to 80 characters long. You can index DsNlabelText using DsNthisPoint.
