get.keyfields.table()
Syntax:

function string  get.keyfields.table  (long object)

Description

Return the name of the table associated with the passed key fields object.

Arguments
long object

ID of the key fields object which must be returned by a previous call to create.keyfields.object()

 

Context

This function is implemented in the 4GL Engine and can be used in all script types.
