set.time.zone()
Syntax:

function long set.time.zone  (string time_zone(50))

Description

This sets the local time zone at run time.

Arguments
string time_zone(50)

 

Return values
0 Success.
-1 Error.

 

Context

This function is implemented in the porting set and can be used in all script types.
