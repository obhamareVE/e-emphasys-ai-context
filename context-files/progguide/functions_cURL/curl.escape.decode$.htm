curl.escape.decode$()
Syntax:

function string curl.escape.decode$  (const string SomeText)

Description

curl.escape.decode$() offers the inverse functionality of curl.escape.encode$().

Arguments
const string SomeText

Pass a string with escape sequences, which need to be replaced.

 

Return values

Contains an string without escape sequences.

Context

This function is implemented in the porting set and can be used in all script types.
