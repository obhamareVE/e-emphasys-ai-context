gbf0001

Currently %d objects.

Do you want to stop further reading ?

Used when end user aborts lengthy read operations. A Yes will stop further reading, a No will continue as if nothing has happened.

Remarks
  • The prefix gbfk means: k eystrokes, gbfs means s ession errors and gbfl is used for all other texts such as: l abels, initial l abel fields (at bottom of window), standard menu l abels, default icon names, and so on.
  • When changing one of more of gbfk0001, gbfk0002 and/or gbfk0003 all other keystroke definitions should also be changed using this new definition, since these messages are used to recognize the keystrokes (see gbf.set.menu.item()).The special keys are defined by the messages gbfk0050 - gbfk0061 (see gbf.set.menu.item()).
  • If gbfl <nr> is a submenu item, then gbfk <nr> (for the same nr of course) is the associated keystroke.
