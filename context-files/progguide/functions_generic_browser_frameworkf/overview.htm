Generic Browser Framework (GBF) overview

The Generic Browser Framework (GBF) provides the developer with a set of functions to easily set up a browser for all kinds of purposes. The browser is capable of displaying a parent-child hierarchy and is based upon the menu browser functionality.

The window created and displayed will look like this:

The default size of the whole window is (width ´ height) 640 ´ 480 pixels, excluding the title bar and status bar. The minimum width of the window is 300 ´ 200 pixels. The five labels in the status bar at the bottom will proportionally increase/decrease in size when the whole window is enlarged/shrunken in horizontal direction. For Baan V the status bar will also contain the latched key indicators (CAPS, NUM, SCRL and KANA).
