Resource sets overview

Resources enable users to change certain display characteristics of their user interface (colors and fonts, for example) without any programming or recompiling. See session ttdsk3160m000 (Runtime resources).

Resources are grouped into resource sets. Resource sets relate to a particular package and user combination. A resource set contains all the resources that a particular application can use. It can also be shared by different applications.

Resource sets are created and maintained using the sessions ttdsk3150m000 (Maintain resource sets) and ttdsk3155m000 (Resource set elements). The sets are stored in the data dictionary.

The resource set functions enable programs to retrieve and set individual resources in resource sets.
