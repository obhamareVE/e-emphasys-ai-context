set.limits.off()
Syntax:

function void  set.limits.off  ()

Description

This removes the limits set with the store.occ.max(), store.occ.min() and/or store.occ.max(), store.occ.min() functions. The entire table can then be accessed again.

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types.
