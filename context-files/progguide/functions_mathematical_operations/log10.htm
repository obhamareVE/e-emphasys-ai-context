log10()
Syntax:

function double  log10  (double x)

Description

This computes the base 10 logarithm of x. The value of x must be greater than zero.

Arguments
double x

 

Return values

The base 10 log of x.

Context

This function is implemented in the porting set and can be used in all script types.
