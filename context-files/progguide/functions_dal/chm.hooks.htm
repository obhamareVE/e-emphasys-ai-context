DAL Change Management hooks

In order to support Workflow Document Authorization, the DAL layer has been extended to support a number of new hooks. These hooks are executed by the Database Change Management (DBCM) layer (see Database Change Management (DBCM) overview).

Note that these hooks should only be implemented in the DAL of the root table of a Business Object.

Available object hooks
  • on.submit()
  • on.recall()
  • on.reject()
  • on.set.draft()
  • on.set.approved()
  • on.set.recalled()
