on.recall()
Syntax:

function long on.recall  ()

Description

This hook is executed in case Workflow Document Authorization is active for the Business Object of which the table of this DAL, is the root table, as defined in the OCM Workflow Model. In case any changes made to the Business Object must be recalled, the DBCM layer will execute this hook. This hook must then publish a Workflow BOD to ION with a request to recall a previous request for approval.

Note: This hook should only be implemented for root tables.

Return values
0

In case of success.
DALHOOKERROR

In case of a failure.

 

Context

This function is implemented in the 4GL Engine and can be used in DAL script types. This function is available from TIV level 1800.
