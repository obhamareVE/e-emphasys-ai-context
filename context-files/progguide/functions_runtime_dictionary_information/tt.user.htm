tt.user()
Syntax:

function boolean  tt.user  (string user(12), ref string name() mb)

Description

This retrieves the name of the specified user ( user) and stores it in the name argument.

Arguments
string user(12)
ref string name() mb

 

Return values

false         error; user not found

true          success

Context

This function is implemented in the 4GL Tools and can be used in all script types.
