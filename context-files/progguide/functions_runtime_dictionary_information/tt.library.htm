tt.library()
Syntax:

function boolean  tt.library  (string library.code, ref string library.desc() mb)

Description

Arguments
string library.code

The library code.
ref string library.desc() mb

This returns the library description.

 

Return values

false         error; library not found

true          success

Context

This function is implemented in the 4GL Tools and can be used in all script types.

Description

This returns information about a specified library.
