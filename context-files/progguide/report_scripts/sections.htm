Report script sections

A report script consists of one or more event sections in which you program actions to be performed at particular states of execution of the report to which the report script is linked. The statements programmed in a report script section consist of a combination of 3GL programming language features: overview and Report script functions.

Report scripts support the following event sections:

  • Program sections (report scripts)
  • Report sections (report scripts)
  • Text field sections (report scripts)
