enable.ue.dll()
Syntax:

function void enable.ue.dll  ()

Description

This macro will enable the calling of ue hooks. The actual state of ue.dll enabling as saved during disable.ue.dll will be used during enable.ue.dll.

Context

This function is implemented in the 4GL Engine and can be used in UEDLL script types.

Example
function extern long ue.before.before.save.object(long mode)
{
        disable.ue.dll()
        db.insert(...)	|Insert on the table of this ue dll
        enable.ue.dll()
        return(0)
}
