214 EUNALLOWEDCOMPNR - Operation not allowed for a logical company
Description:
This error indicates that an operation for mapping company numbers is not allowed. If the logical company is not equal to the physical company, you are not allowed to do a drop/clear table operation.
Solution:
Ensure the table action is not executed for a logical company, either by not executing it or by executing it for the related physical company.
