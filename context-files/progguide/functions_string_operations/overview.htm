String operations overview

Use these functions to manipulate strings in various ways. For example:

  • to check whether a string contains the textual representation of an integer value
  • to check whether a string contains spaces only
  • to check the number of characters or bytes in a string
  • to convert the byte encoding of a numerical value, stored in a string, to the encoded numerical value
  • to convert the textual representation of a numerical value to the represented numerical value
  • to convert a numerical value to its textual representation
  • to convert a string to upper case or lower case
  • to search for a substring within a string
  • to strip leading and trailing spaces from strings
  • to align strings
  • to convert a numerical value to its byte encoding and store it in a string.
