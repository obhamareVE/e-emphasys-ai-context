Overview

This "Gantt and Schedule chart" API is a replacement for the old Plan Chart Manager API. The old Plan Chart Manager supports BW only. This new API supports WebTop only, since WebTop is positioned as the preferred UI and BW will be phased out in the future. The minimal WebTop version required is WebTop 4.1.

This new API is easier to work with than the old Plan Chart Manager API. The old API is very low-level, whereas this API is more high-level.

For the end-user the difference is also big. The Gantt and Schedule charts have a modern-looking, easy-to-work-with interface.
