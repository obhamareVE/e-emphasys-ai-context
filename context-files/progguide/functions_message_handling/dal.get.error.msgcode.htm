dal.get.error.msgcode()
Syntax:

function string  dal.get.error.msgcode  ()

Description

This retrieves the message code of most recent DAL error message from the message buffer. The message is not removed from the buffer.

Return values

The message code.

Context

This function is implemented in the 4GL Tools and can be used in DAL script types.
