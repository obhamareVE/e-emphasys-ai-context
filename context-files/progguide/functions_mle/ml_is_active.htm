ml_is_active()
Syntax:

function boolean  ml_is_active  ()

Description

Tests whether Multi Language support is active on runtime for the current environment.

Return values
TRUE Multi Language support is active.
FALSE Multi Language support is inactive.

 

Context

This function is implemented in the 4GL Tools and can be used in all script types.
