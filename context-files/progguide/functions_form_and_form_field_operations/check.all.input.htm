check.all.input()
Syntax:

function void check.all.input  ()

Description

This executes all check.input subsections (if any) included in the program script. When an input error is detected, the choice.again() function is automatically executed. The check.all.input function doesn't work for array fields.

When a DAL is used, the property checks in the DAL are executed after the check.input subsections of the UI script are executed.

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types.

Example
choice.cont.process:
before.choice:
check.all.input()
