get.prev.value()
Syntax:

function boolean  get.prev.value  (long value)

Description

This function retrieves the field's previous value. The previous value is converted to the type of the output parameter.

This function only works on the current field

Arguments
long value

 

Return values

true when successfull, otherwise false.

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Example
asd
