to.satellite()
Syntax:

function void to.satellite  (long selected.tab)

Description

This function can be used in UI scripts for multi main table controller sessions to switch to another satellite session. It can be used after the form definition has been read; it cannot be used in the before.program section.

Arguments
long selected.tab

The selected tab:

when numeric: the index of the tab to select.

when alfabetic: the name of the satellite session.

 

Context

This function is implemented in the 4GL Engine and can be used in 4GL script types.

Note

This function is available from Tools Interface Version (TIV) TIV 1900.
