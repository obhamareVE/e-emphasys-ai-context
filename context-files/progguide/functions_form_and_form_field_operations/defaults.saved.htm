defaults.saved()
Syntax:

function boolean  defaults.saved  ()

Description

Determines if there are saved defaults for the current user/company/session combination.

Return values

FALSE      user defaults not found

TRUE        user defaults found

Context

This function is implemented in the 4GL Engine and can be used in all script types.
