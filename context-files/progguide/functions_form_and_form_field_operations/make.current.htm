make.current()
Syntax:

function void  make.current  ()

Description

This reads the current main table record and displays it on screen. Calling this function is similar to calling execute(find.data), except that if the specified record is not present, make.current() does not read the next record.

Context

This function is implemented in the 4GL Engine and can be used in all script types.
