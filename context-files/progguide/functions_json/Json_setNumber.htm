Set a JSon number object
Syntax:

function long Json.setNumber  (long json, const string key, ... value)

Description

Convenience function for directly associating a 3GL long or double value with a key in a JSON object. The 3GL long or double value is first converted to a JSON number value which is then associated with the specified key. The function is a shorthand for: Json.set(json, key, Json.newNumber(value))

Arguments
long json

json is a JSON object.

const string key

key is a key to associate the JSON number value with.

... value

value the 3GL long or double value for which a JSON number value is created, which is then associated with the key.

 

Return values

Optional, the new JSON number value which has been associated with the key.

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Preconditions
  • Parameter 'json' is a JSON value of type JSON_TYPE_OBJECT.
Note
  • We have to use the ... notation and work with variable arguments; the reason is that the compiler complains when passing a LONG value to a DOUBLE argument
