Add a string object to a JSon object
Syntax:

function long Json.addString  (long json, string value)

Description

Convenience function for directly adding a 3GL string value to a JSON array. The 3GL string value is first converted to a JSON string value which is then added to the array. The function is a shorthand for: Json.add(json, Json.newString(value))

Arguments
long json

json is a JSON array.

string value

value is the 3GL string value for which a JSON string value is created, which is then added to the array.

 

Return values

Optional, the new JSON string value which has been added to the JSON array.

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Preconditions
  • Parameter 'json' is a JSON value of type JSON_TYPE_ARRAY.
