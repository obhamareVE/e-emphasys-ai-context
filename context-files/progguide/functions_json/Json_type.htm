Get the JSon object type
Syntax:

function long Json.type  (long json)

Description

Gets the type of a JSON value.

Arguments
long json

json is a JSON value.

 

Return values

The type of the JSON value. One of { JSON_TYPE_OBJECT, JSON_TYPE_ARRAY, JSON_TYPE_BOOLEAN, JSON_TYPE_NUMBER, JSON_TYPE_STRING, JSON_TYPE_NULL, JSON_TYPE_UNKNOWN }.

Context

This function is implemented in the 4GL Engine and can be used in all script types.
