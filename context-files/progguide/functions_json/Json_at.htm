Return a JSon object in a JSon array
Syntax:

function long Json.at  (long json, long idx)

Description

Returns the JSON value that is stored at the specified index in a JSON array.

Arguments
long json

json is a JSON array.

long idx

idx is the index in the array to get the value from.

 

Return values

The JSON value that is stored in the JSON array at the specified index.

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Preconditions
  • Parameter 'json' is a JSON value of type JSON_TYPE_ARRAY.
  • Parameter 'idx' is greater than 0 and less than or equal to the number of values in the array.
