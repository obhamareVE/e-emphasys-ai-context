Write a JSON value object to a file as JSON text
Syntax:

function long Json.writeFile  (long json, const string path [, long options])

Description

Writes a JSON value object to a file as JSON text. By default:

  • The JSON text is converted to UTF-8.
  • Extra new-lines and indents are added to make the output human readable.

Arguments
long json

json is a JSON value.

const string path

path is the path to the file to write the JSON text to; if the file already exists it is overwritten.

[long options]

options are the options to use when writing the JSON text; the following options can be specified:

  • JSON_WRITE_UTF8 - converts the JSON text to UTF-8.
  • JSON_WRITE_PRETTY - adds new-lines and indents to make the output human readable.
  • JSON_WRITE_COMPACT - no whitespace is added between JSON tokens.

By default, the file is written in pretty mode and in UTF-8 format.

 

Return values

>= 0 the number of bytes written to the string when successful

< 0 in case of a failure [e.g. when the string to write to is too small]

variable 'e' contains an error code [e.g. ENOSPC]

Context

This function is implemented in the 4GL Engine and can be used in all script types.

Preconditions
  • Parameter 'json' is a JSON value.
