42I14 - Invalid reference path element

An SQL statement contains a REFERS TO predicate with a PATH clause and this PATH clause contains a PATH element that is not of the form

Example
select *
from   dbtst160, dbtst120
where  dbtst160.deptno refers to dbtst120 path dbtst100

The PATH element is missing a column name. This query will cause the error:

SQLState 42I14: Invalid reference path element 'dbtst100'
