42I93 - Invalid SQL delete

The table contains MLFs (columns with multi language data). SQL delete does not delete the translations in the non-base language, so SQL delete is now allowed.

Example
	      delete from dbmlf120 where nr > 100

When the table dbmlf120 contains an MLF this query will cause the error:

SqlState=42I93, 'Table 'dbmlf120' has a shadow table that needs to be updated on delete'
