42I89 - Invalid company number expression

The company number expression is not valid.

Examples
select *
from dbtst190 a 
   left join dbtst120 b on a.empno = b.empno
      left join dbtst100 c on b.workdept = c.deptno and a.company_nr = b.company_nr

This query will cause the following error message:

SqlState=42I89: company_nr join predicate on columns 'a.company_nr' and 'b.company_nr' does not restrict the company set of the right branch of the LEFT OUTER JOIN 

select *
from dbtst120
where company_nr in ( 0,1 ) and company_nr in ( 0,1 )

This query will cause the following error message:

SqlState=42I89: Multiple company_nr IN predicates defined within a query specification
