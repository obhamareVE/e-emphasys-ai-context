42I66 - Table has a parent table that needs to be updated on delete

The table in the FROM clause of an SQL DELETE statement has a parent table; the reference mode of this parent-child relationship is mandatory or mandatory unless empty and the reference is maintaind through reference counters.

Example
delete from dbtst146

There is a reference from dbtst146.empno.mu to dbtst120.empno. The reference mode is mandatory unless empty and is maintained through counters. This means that this delete would require that the reference counters of table dbtst120 are updated. An SQL DELETE statement can only be executed if no update actions on a parent table are required. This statement will cause the error:

SQLState 42I66: Table 'dbtst146' has a parent table that needs to be updated on delete
