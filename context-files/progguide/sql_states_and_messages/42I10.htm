42I10 - Combined column used in expression

An SQL statement contains an expression with a binary operator (for example, addition or subtraction) and one of the operands of this binary operator is a combined column. This error can be caused by any of the binary operators and will occurr even if the combined column consist of just one child column.

Example
select *
from   dbtst120
where  cmba / 10 > 1

The column cmba is a combined column that contains one child column edlevel. This query will cause the error:

SQLState 42I10: Combined column 'cmba' used in expression
