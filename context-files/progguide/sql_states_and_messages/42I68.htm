42I68 - Update on foreign key column

An SQL UPDATE statement attempts to update a foreign key column.

Example
update dbtst120
set workdept = ''

Column workdept is a foreign key column. An SQL UPDATE statement cannot be used to update a foreign key. This statement will cause the error:

SQLState 42I68: Update on foreign key column 'workdept'
