42I64 - Array column used in SET clause

An SQL UPDATE statement contains an array column in its SET clause

Example
update dbtst120
set    phoneno = "0342-123456"

Column phoneno is an array column of strings. This statement will cause the error:

SQLState 42I64: Array column 'phoneno' used in SET clause

Array columns cannot be updated using an SQL UPDATE statement.
