42I60 - FROM clause not found where expected

A query or subquery does not have a FROM clause, or it does have a FROM clause, but due to some other syntax error the FROM keyword is not found at the position in the statement where it is expected.

Example 1
select dbtst120.firstnme
where  dbtst120.empno  = 10

This query does not have a FROM clause. The statement contains the keyword WHERE at the position where the keyword FROM is expected.

SQLState 42I60: FROM clause not found where expected. Found 'where' instead.

Example 2
select empno firstnme
from   dbtst120

There is a comma missing after column empno. Because of this, the keyword FROM is expected right after empno, but firstnme is found instead. This query will cause the error:

SQLState 42I60: FROM clause not found where expected. Found 'firstnme' instead.
