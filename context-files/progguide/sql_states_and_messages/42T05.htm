42T05 - Invalid type for operand of LIKE operator

An SQL statement contains a LIKE predicate and the left operand of this LIKE predicate is not of type string.

Example
select empno
from   dbtst120
where  edlevel like "1.*"

Column edlevel is of type integer. The left operand of a LIKE predicate may only be of type string. This query will cause the error:

SQLState 42T05: Invalid type for left operand of LIKE operator. Type STRING expected
