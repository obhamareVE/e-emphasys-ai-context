42I25 - SELECT clause of subquery contains more than 1 item

An SQL statement contains a subquery that selects more than one column where it is expected to select just a single column.

Example 1
select firstnme
from   dbtst120
where  empno = ( select mgrno, deptname
                 from   dbtst100
                 where  deptno = "A01" )

A subquery in a comparison predicate may only select one column. This query causes the error:

SQLState 42I25: SELECT clause of scalar subquery contains more than 1 item

Example 2
select firstnme
from   dbtst120
where  empno IN ( select mgrno, deptname
                  from   dbtst100 )

The subquery of an IN predicate may only select one column. This query causes the error:

SQLState 42I25: Select clause of subquery in IN predicate contains more than 1 item
