42I05 - The value of the numeric literal is out of range

An SQL statement contains an integer or real constant that is outside the range of possible values.

Example
select *
from   dbtst120
where  edlevel > 9999999999

Column edlevel is of type integer and the maximum integer value is 2147483647. This query will cause the error:

SQLState 42I05: The value of the numeric literal '9999999999' is out of range
