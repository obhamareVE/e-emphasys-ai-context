42I52 - Result parameter defined on multi part combined column

An SQL statement contains a select target for a combined column that contains more than one child column.

Example 1
select cmbb :earnings
from   dbtst120

The column cmbb is a combined column that constains two child columns: salary and bonus. The select target earnings is used for cmbb, but because only one value can be selected in the select target, this query will cause the error:

SQLState 42I52: Result parameter defined on multi part combined column 'cmbb'
