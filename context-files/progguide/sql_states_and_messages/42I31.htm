42I31 - Multiple compnr specifications on table

An SQL statement contains two or more company number specifications for the same table.

Example
select *
from   dbtst120
where  _compnr = 100 and _compnr = 200

This query causes the error:

SQLState 42I31: Multiple compnr specifications on table 'dbtst120'

You can use a UNION query to retrieve data from multiple companies: 

select *
from   dbtst120
where  _compnr = 100
union all
select *
from   dbtst120
where  _compnr = 200
