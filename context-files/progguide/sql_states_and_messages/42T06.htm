42T06 - Invalid type for operand of substring operator

An SQL statement contains a substring expression and one of the operands of this substring expression is not of the correct type.

Example 1
select empno
from   dbtst120
where  edlevel(1;1) = "1"

Because column edlevel is of type integer, this query will cause the error:

SQLState 42T06: Invalid type for operand of substring operator. Expected type STRING

Example 2
select firstnme
from   dbtst120
where  firstnme(1;bonus) = "J"

The start and length operands of a substring expression must be of type integer. Column bonus is of type real. This query will cause the error:

SQLState 42T06: Invalid type for operand of substring operator. Expected type INTEGER
