22026 - String data length mismatch

A raw value is assigned to a raw column and the length of the value is smaller than the length of the column.

Example
update dbtst230 a set a.raw = x'0'
where a.empno = 10

Column is a raw of length 20, and the value assigned to it is shorter than 20 bytes. This will result in the following error:

SqlState=22026, 'Value cannot be assigned to column dbtst230.raw', NativeError=205

It can be solved by assuring that the value assigned to the raw column is of the same length as the column. 

update dbtst230 a set a.raw = x'CAFEBABECAFEBABECAFEBABECAFEBABECAFEBABE'
where a.empno = 10
