42I04 - Unknown table

An SQL statement contains a reference to a table that does not exist or that is not known in that particular scope.

Example 1
select deptno
from   dbtst121

Table dbtst121 does not exist. That is, the runtime datadictionary does not contain a table called dbtst121. This query will cause the error:

SQLState 42I04: Unknown table 'dbtst121'

If the table does exist in the runtime datadictionary, but it does not exist in the RDBMS, then you will get the error:

SQLState QP999: Table does not exist

Example 2
select dbtst120.salary
from   dbtst120 a

Table dbtst120 uses alias 'a'. Therefore, the tablename dbtst120 is not known and the select item dbtst120.salary causes the error:

SQLState 42I04: Unknown table 'dbtst120'
