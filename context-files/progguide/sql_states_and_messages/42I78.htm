42I78 - Invalid timestamp string

An invalid timestamp string literal is specified.

Example
select TIMESTAMP 'Tue Nov 28 11:49:17 NFT 2006'
from   dbtst120

The format of the timestamp string literal is not correct. This query causes the error:

SQLState 42I78: Invalid timestamp string Tue Nov 28 11:49:17 NFT 2006

The proper format is 'YYYY-MM-DD hh:mm:ss':

select TIMESTAMP '2006-11-28 11:49:17'
from   dbtst120
