42I11 - Combined column used in combined

An SQL statement contains an explicit row value constructor (also called a Comnbined) and one of the elements of this row value constructor is a combined column.

Example
select *
from   dbtst120
where  { cmbb } > { 50000, 1000 }

The column cmbb is a combined column that contains two child columns salary and bonus. This query will cause the error:

SQLState 42I11: Combined column 'cmbb' used in combined
