42T20 - Invalid argument type for AVG/SUM function

An SQL statement contains a SUM or AVG set function and the operand of this set function does not have a numeric type.

Example 1
select sum(firstnme)
from   dbtst120

The set function SUM can only be applied to operands of type integer or real. Column firstnme is of type string. This will cause the error:

SQLState 42T20: Invalid argument type for SUM function

Example 2
select avg(hiredate)
from   dbtst120

The set function AVG can only be applied to operands of type integer or real. Column hiredate is of type date. This will cause the error:

SQLState 42T20: Invalid argument type for AVG function
