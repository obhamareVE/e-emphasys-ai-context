42I70 - Reference FROM clause used with non-constant reference

An SQL statement contains a REFERS TO predicate with a FROM clause and this REFERS TO FROM clause is applied to a column and not to a constant.

Example
select empno, deptname
from   dbtst120, dbtst100
where  workdept from dbtst160.deptno refers to dbtst100

The FROM clause dbtst160.deptno of the REFERS TO predicate is applied to the column workdept. A REFERS TO FROM clause can only be applied to a constant. This query will cause the error:

SQLState 42I70: Reference FROM clause 'dbtst160.deptno' used with non-constant reference
