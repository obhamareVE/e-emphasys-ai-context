42I42 - Multiple 'no hints' hints specified

An SQL statement contains a HINT clause that contains more than one 'no hints' hint.

Example
select *
from   dbtst120
hint   no hints
and    no hints

This query causes the error:

SQLState 42I42: Multiple 'no hints' hints specified
