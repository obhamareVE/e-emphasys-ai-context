42I29 - Compnr specification in NOT condition

An SQL statement contains a company number specification that is contained in a NOT condition.

Example
select *
from   dbtst120
where  not ( _compnr = 100 )

This query causes the error:

SQLState 42I29: Compnr specification in NOT condition
