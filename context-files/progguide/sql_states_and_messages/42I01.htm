42I01 - Duplicate table name or alias name

A FROM clause in an SQL statement contains two or more tables that have the same exposed name. The exposed name of a table is simply the table name, unless the table has an alias. In that case the alias is the exposed name.

Example 1
select *
from   dbtst120, dbtst120

Table dbtst120 appears twice in the FROM clause. This query will cause the error:

SQLState 42I01: Duplicate table name or alias name 'dbtst120'

It can be solved by giving one of the tables an alias so that the tables have a unique exposed name: 

select *
from   dbtst120, dbtst120 tbl120
Example 2
select *
from   dbtst120, dbtst100 dbtst120

Due to the alias of table dbtst100, both tables in the FROM clause have the same exposed name. This query will cause the error:

SQLState 42I01: Duplicate table name or alias name 'dbtst120'
