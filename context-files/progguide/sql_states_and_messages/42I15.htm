42I15 - REFERS TO predicate in OR condition

An SQL statement contains a REFERS TO predicate that is contained in an OR condition.

Example
select *
from   dbtst120, dbtst100
where  empno > 100 or ( empno < 100 and workdept refers to dbtst100 )

This query causes the error:

SQLState 42I15: REFERS TO predicate in OR condition
