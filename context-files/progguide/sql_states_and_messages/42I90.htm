42I90 - Invalid set function specification

The set function specification is not valid.

Example
select max(max(salary))
from dbtst120

This query will cause the error:

SQLState 42I90: Set function specification used in set function specification

select max((select salary from dbtst120))
from dbtst120

This query will cause the error:

SQLState 42I90: Subquery used in set function specification

select edlevel
from dbtst120
group by edlevel
having exists( select avg(salary) from dbtst190 )

This query will cause the error:

SQLState 42I90: Outer column reference 'salary' used in set function specification
