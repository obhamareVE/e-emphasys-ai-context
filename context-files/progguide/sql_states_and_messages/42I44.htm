42I44 - Invalid index number

An SQL statement contains an index hint that specifies an index number that is not an existing index.

Example
select *
from   dbtst120
where  edlevel > 10
hint   use index 3 on dbtst120

The table dbtst120 has only two indexes. This query causes the error:

SQLState 42I44: Invalid index number 3 for table 'dbtst120'
