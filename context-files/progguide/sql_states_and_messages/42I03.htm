42I03 - Unknown column

An SQL statement contains a reference to a column that does not exist or that does not exist in that particular scope.

Example 1
select deptno
from   dbtst120

Table dbtst120 does not contain a column with the name deptno. This query will cause the error:

SQLState 42I03: Unknown column 'deptno'

Example 2
select dbtst100.mgrno
from   dbtst120
where  empno in ( select mgrno from dbtst100 )

Table dbtst100 does contain a column mgrno, but this column is not in the scope of the main query. This query will cause the error:

SQLState 42I03: Unknown column 'dbtst100.mgrno'
