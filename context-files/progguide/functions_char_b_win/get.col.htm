get.col()
Syntax:

function long get.col  ()

Description

Deprecated. This function is only supported for character-based windows and its usage is therefore deprecated.

This retrieves the number of the column where the cursor is currently positioned. The function uses get.cp(), so it is slower than that function.

Context

This function is implemented in the porting set and can be used in all script types.
